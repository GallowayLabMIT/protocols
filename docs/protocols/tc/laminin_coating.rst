========================
Laminin Coating 
========================

Stocks and supplies
---------------------------------------------------------

    - 100x Laminin (1 mg/mL; stored -20 deg C)
    - PBS

Protocol
-------------------------------------------------------

1. Add laminin to PBS. 
2. Add 2 mL laminin-PBS to each 6 well plate. 
3. Allow wells to set for 2-4 hours before removing excess.
